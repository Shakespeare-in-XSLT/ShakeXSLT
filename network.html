Network Analysis

Representation of the characters' relationships

Network analysis (NA) is a set of integrated techniques to depict relations among actors and to analyze the social structures that emerge from the recurrence of these relations.
To stress the importance of the relationships between the characters, which are the center of gravity of a play's structure, we have decided to implement three different graphs for this purpose. By clicking on them, it is possible to isolate the relationships' network of the current node (the picked character).
